# NAKAMA

## Pre-requirements
+ Java 8
+ NodeJS
+ MySQL with schema 'nakama'

## Install
### NodeJS dependencies
#### Install
    npm install
    npm install webpack -g
#### Update
    npm update
### Node Library
    gradlew copyNodeLibs
## Run
### Spring Boot
#### Gradle
    gradlew bootRun
#### Java
Run Application.java
### Tomcat WAR
Add project into Tomcat
