# NAKAMA

## Pre-requirements
+ Java 8
+ NodeJS
+ MySQL with schema 'nakama'

## Install Steps
    npm install
    npm install webpack -g
    npm webpack
    ./gradlew copyNodeLibs

## Run
### Spring Boot
#### Gradle
    gradlew bootRun
#### Java
Run Application.java
### Tomcat WAR
Add project into Tomcat

## Application credentials
admin / admin
